# تطبيق جرد صيدلية — Al-Ghaith Pharmacy (Firebase)

هذا المشروع نسخة مدمجة مع Firebase Realtime Database و Firebase Storage.
الوظائف:
- إضافة مادة (اسم، صنف، تاريخ انتهاء، كمية، ملاحظة، صورة بواسطة الكاميرا الخلفية).
- كل موظف له جدول جرد منفصل (ايهاب، بسام، ضحى، سارة، حوراء، معاذ، محمود، نزار).
- قسم "نواقص المسواك" مع إمكانية تصوير وكتابة ملاحظة.
- الصور مخزنة في Firebase Storage، والبيانات في Realtime Database.
- التحديثات تظهر فورياً على جميع الأجهزة المتصلة.

التشغيل:
1. ضع الملفات `index.html`, `styles.css`, `app.js` و `README.md` في مجلد واحد.
2. افتح `index.html` في متصفح حديث (يفضل تشغيل على HTTPS على الهاتف لاستخدام الكاميرا بشكل موثوق).
3. عند إضافة صورة سيتم رفعها إلى Firebase Storage وتخزين رابطها في قاعدة البيانات.

ملاحظات أمان:
- إعدادات `firebaseConfig` مرئية في الكود لأن SDK للعمل في الواجهة الأمامية يحتاجها. ضع قواعد أمان مناسبة في Firebase Console (قواعد قاعدة البيانات وStorage) قبل الاستخدام في الإنتاج.
- حالياً يفترض التطبيق وصولاً مفتوحاً للقراءة/الكتابة (إن تطلبت، سأقوم بإعداد مثال لقواعد أمان).

تحسينات مقترحة لاحقاً:
- إضافة تسجيل دخول للموظفين (Authentication) لربط كل عملية بملف المستخدم.
- إنشاء واجهة تعديل منفصلة بدلاً من حذف/إعادة إضافة.
- إضافة تصغير الصور قبل الرفع لتقليل استهلاك التخزين وحجم التحميل.
- واجهات تقرير أو تصدير CSV.
